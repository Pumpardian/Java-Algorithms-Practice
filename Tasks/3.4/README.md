#Task 32

Хеш-атака. Найдите 2^n строк, каждая длиной 2^n, которые имеют одинаковое значение hashCode() для следующей реализации:

public int hashCode()
{
	int hash = 0;
	int skip = Math.max(1, length()/8);
	for (int i = 0; i < length(); i += skip)
		hash = (hash * 37) + charAt(i);
	return hash;
}
